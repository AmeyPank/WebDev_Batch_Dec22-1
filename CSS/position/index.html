CSS Positions

  1. Static: The original or default position of an element.
  2. Relative:Positions an element relative to it's normal/original position. In this case, we can add offset values to move the element. Element stays in the normal document flow.
  3. Absolute: Element is removed from the normal document flow, no space is created for that element. Positions relative to its parent. If no parent present then positions itself relative to the main document.
  4. Fixed: Elements are positioned relative to the viewport. Element does not move and stays in the same position even on scrolling. Element is removed from normal flow.
  5. Sticky: Element is positioned according to normal document flow. Till the element hits the top, it has normal position, after it touches the top, then it's position becomes fixed.
